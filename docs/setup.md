# Setup

## 1. Create the Folder Structure

Inside your Obsidian vault, create:

```txt
0-Inbox/
1-Templates/
2-Notes/
3-Media/
```

Use these folders as workflow trays:

- `0-Inbox`: unprocessed captures.
- `1-Templates`: reusable templates.
- `2-Notes`: processed knowledge.
- `3-Media`: files and attachments.

## 2. Copy Templates

Copy the files from `templates/` into `1-Templates/`.

If you use Obsidian core Templates, set the template folder to `1-Templates`.

## 3. Create a Home Dashboard

Copy `examples/home-dashboard.md` into the root of your vault as `Home.md`.

This page gives you quick access to:

- Inbox;
- Areas;
- active deadlines;
- notes needing processing;
- weekly review.

## 4. Create an Areas MOC

Copy `examples/areas-moc.md` into `2-Notes/Areas.md`.

Then adapt the area list to your life and work. Keep it small at the beginning.

## 5. Install Optional Plugins

Recommended:

- Dataview
- Tasks
- Templates or Templater
- Obsidian Git

Optional, depending on your workflow:

- Excalidraw for visual thinking;
- Calendar if daily notes are central for you;
- Style Settings if you use a theme that supports it.

## 6. Weekly Review

Every week:

1. Empty Inbox.
2. Delete low-value captures.
3. Promote useful captures into `2-Notes`.
4. Add links to existing MOCs.
5. Create a new MOC only when navigation improves.
6. Update next actions in projects and ideas.

A second brain decays when review disappears.
