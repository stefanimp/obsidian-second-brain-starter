## Concepts

- MOC: Map of Content, a curated index note used to navigate a topic.
- Active recall: retrieving knowledge from memory instead of only rereading.
- Progressive summarization and second brain workflows popularized by Tiago Forte.

## Video Inspiration

The workflow is also compatible with the idea discussed in the video "Obsidian: The King of Learning Tools (FULL GUIDE + SETUP)": notes should be written in your own words, links can replace many tag use cases, and folders should not be the whole architecture.
